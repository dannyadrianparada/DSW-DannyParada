Flex Box

Definición

Flexbox es un método de diseño de página unidimensional para compaginar elementos en filas o columnas. Los elementos de contenido se ensanchan para rellenar el espacio adicional y se encogen para caber en espacios más pequeños.
En otras palabras, Flexbox es una forma más fácil de organizar los objetos dentro de un contenedor. Además, tiene la ventaja de que permite que dichos objetos se acomoden automáticamente al ancho y alto de la pantalla.

Los dos ejes Flexbox

Todo lo que se hace con Flexbox, está basado en dos ejes. El eje principal (definido por "flex-direction"), y el eje cruzado. la propiedad flex-direction puede tener 4 valores: row, row-reverse, column y column-reverse.
Si el valor es row o row-reverse, entonces el eje principal correrá en dirección horzontal de la página. Pero si el valor es column o column-reverse, entonces el eje principal correrá en la direcicón vertical de la página. Por su parte, el eje cruzado siempre irá perpendicular al eje principal.

El Contenedor

Para que un objeto lo podamos manejar con flexbox, es necesario que dicho objeto esté dentro de un contenedor padre, al cual se le haya aplicado la propiedad display-flex.

Lístas

Las listas en HTML nos permite crear conjuntos de elementos en forma de lista dentro de una página, todos los cuales irán precedidos, generalmente, por un guión o número. Hay dos tipos de listas principales. Las listas numeradas y las no numeradas.

Para crear una lista numerada en html, debes usar la etiqueta "ol" junto con la etiqueta "li", la cual conforma cada item de la lista.

Para crear una lista no numerada, se usa la etiqueta "ul" junto con "li"

Enlaces

Lo más importante de los documentos HTML son los enlaces, Ya que mediante los enlaces en HTML podemos comunicar una página con otra. De esta forma, enlazando documentos HTML podemos acabar tejiendo lo que es Internet.

Para crear un enlace en un documento web, se usa la etiqueta "a", la cual tiene la propiedad href, que es donde debes colocar la dirección de la página a la quieres ir.
